# Smart Petrol Cost Calculator

Smart Petrol Cost Calculator is an Android application developed using Android Studio. 
The application helps users calculate petrol costs based on petrol type, petrol price, fuel usage, and BUDI MADANI eligibility.

Features:
- Select Petrol Type (RON95, RON97, Diesel)
- Enter Petrol Price per Liter
- Enter Fuel Usage
- Select BUDI MADANI Eligibility
- Calculate Total Petrol Cost
- Display Rebate Amount
- Display Final Payable Amount
- About Page with developer information

Calculation Formula:
Total Petrol Cost = Petrol Price × Fuel Usage
BUDI Rebate = Fuel Usage × RM1.99
Final Payable Amount = Total Petrol Cost − Rebate
